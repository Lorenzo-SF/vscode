# Atajos de Teclado - ElixIDE

## Atajos Generales

### Navegación
- `Ctrl+Shift+E` / `Cmd+Shift+E` - Mostrar/Ocultar Explorer
- `Ctrl+Shift+F` / `Cmd+Shift+F` - Mostrar/Ocultar Search
- `Ctrl+Shift+G` / `Cmd+Shift+G` - Mostrar/Ocultar Source Control
- `Ctrl+Shift+D` / `Cmd+Shift+D` - Mostrar/Ocultar Run and Debug
- `Ctrl+Shift+X` / `Cmd+Shift+X` - Mostrar/Ocultar Extensions

### Action Bar en Titlebar
- `Tab` - Navegar entre botones de la action bar
- `Shift+Tab` - Navegar hacia atrás
- `Enter` / `Space` - Activar botón enfocado
- `Arrow Left/Right` - Navegar entre botones (cuando está enfocado)

### Comandos ElixIDE
- `Ctrl+Shift+P` / `Cmd+Shift+P` - Abrir Command Palette
  - "ElixIDE: Export Logs" - Exportar logs para diagnóstico
  - "ElixIDE: Show Version" - Mostrar versión de ElixIDE

## Atajos del Editor

Los atajos estándar de VSCode funcionan en ElixIDE:

- `Ctrl+N` / `Cmd+N` - Nuevo archivo
- `Ctrl+O` / `Cmd+O` - Abrir archivo
- `Ctrl+S` / `Cmd+S` - Guardar
- `Ctrl+W` / `Cmd+W` - Cerrar editor
- `Ctrl+F` / `Cmd+F` - Buscar
- `Ctrl+H` / `Cmd+H` - Reemplazar
- `Ctrl+/` / `Cmd+/` - Toggle comentario

## Personalización

Los atajos de teclado pueden personalizarse en:
- **File > Preferences > Keyboard Shortcuts** (Windows/Linux)
- **Code > Preferences > Keyboard Shortcuts** (macOS)
