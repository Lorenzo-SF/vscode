# Guía de Inicio Rápido de ElixIDE

## Instalación

### macOS
1. Descargar `ElixIDE-{version}-darwin-{arch}.dmg`
2. Abrir el archivo .dmg
3. Arrastrar ElixIDE a la carpeta Applications
4. Abrir ElixIDE desde Applications

### Linux
1. Descargar `ElixIDE-{version}-x86_64.AppImage` o `.deb` / `.rpm`
2. Para AppImage: `chmod +x ElixIDE-{version}-x86_64.AppImage && ./ElixIDE-{version}-x86_64.AppImage`
3. Para .deb: `sudo dpkg -i elixide-{version}-amd64.deb`
4. Para .rpm: `sudo rpm -i elixide-{version}.x86_64.rpm`

### Windows
1. Descargar `ElixIDE-{version}-win32-x64.exe`
2. Ejecutar el instalador
3. Seguir las instrucciones del instalador
4. Abrir ElixIDE desde el menú de inicio

## Primera Ejecución

1. Abrir ElixIDE
2. Seleccionar un tema (ElixIDE Dark es el predeterminado)
3. Abrir un proyecto Elixir existente o crear uno nuevo
4. Disfrutar de las funcionalidades específicas de Elixir

## Características Principales

- **Action Bar en Titlebar**: Los botones de actividad están en la barra superior
- **Temas ElixIDE**: Temas oscuro y claro con colores inspirados en Elixir
- **Soporte para Elixir**: Resaltado de sintaxis específico para Elixir
