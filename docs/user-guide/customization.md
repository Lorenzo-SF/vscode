# Personalización de ElixIDE

## Temas

ElixIDE incluye dos temas predeterminados:

### ElixIDE Dark
- Tema oscuro con paleta de colores inspirada en Elixir
- Fondos morados oscuros
- Acentos lavanda y morado claro
- Predeterminado para nuevas instalaciones

### ElixIDE Light
- Tema claro con paleta de colores inspirada en Elixir
- Fondos beige claros
- Acentos morados

### Cambiar Tema
1. File > Preferences > Theme > Color Theme
2. O usar `Ctrl+K Ctrl+T` / `Cmd+K Cmd+T`
3. Seleccionar "ElixIDE Dark" o "ElixIDE Light"

## Action Bar

La action bar está ubicada en la barra superior (titlebar) y contiene:
- Explorer (Explorador de archivos)
- Search (Búsqueda)
- Source Control (Control de código fuente)
- Run and Debug (Ejecutar y depurar)
- Extensions (Extensiones)

Los botones se pueden mostrar/ocultar mediante la configuración `workbench.activityBar.visible`.
